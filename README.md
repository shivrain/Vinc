## Tech notes

- **No build step.** Pure HTML + Tailwind CDN + Google Fonts.
- **Single file.** All 23 slides + navigation JS embedded in one `index.html`.
- **Responsive scaling.** Slides authored at 1280×720; the runtime scales each to fit the viewport while preserving aspect ratio.
- **Dependencies (loaded from CDN at runtime):**
  - Tailwind CSS — `https://cdn.tailwindcss.com`
  - Google Fonts — Inter + Playfair Display

No external JS frameworks. No bundler. Works offline once Tailwind + fonts are cached.

## Rebuilding from individual slides

The presentation was assembled from `../Presentation/slide_*.html`. If you edit an individual slide, regenerate `index.html`:

```bash
python3 build.py
```

(See `build.py` in this folder.)
